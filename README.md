## Computational Physics II

🔒 Part of this project is private.

This repository contains codes, simulations, and numerical projects developed during the **Computational Physics II** course.

The focus is on more advanced computational techniques applied to physical systems, including numerical solutions of complex differential equations, simulations of dynamical systems, and computational modeling of physical phenomena.

Furthermore, the exercises in Newman's book are open. The problem sets are closed, but will be gradually opened as the course progresses.
